# Tech Stack

## Core Technologies

- **PHP**: 8.4 FPM
- **Web Server**: Nginx (Alpine)
- **Framework**: Symfony 6.4 LTS
- **Database**: PostgreSQL 17 (Latest)
- **ORM**: Doctrine ORM
- **Containerization**: Docker & Docker Compose
